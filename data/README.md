# Data

The included train/validation/test files are synthetic spectra generated from smooth basis functions plus random absorption bands.

Columns:

- `sample_id`
- `source`
- `color_name`
- `L`, `a`, `b`
- `r_380`, `r_390`, ..., `r_780`

The files are intended for pipeline validation, not scientific claims. Replace or augment them with public spectral datasets for real experiments.
