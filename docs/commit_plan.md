# Commit Plan

Use small commits that tell a pretraining research story.

1. `Add synthetic spectral data generator and train/val/test splits`
2. `Add masked spectral transformer pretraining baseline`
3. `Add DeltaE and reconstruction evaluation harness`
4. `Add objective ablation configs and sweep runner`
5. `Add scaling configs and result summarization`
6. `Add plotting and failure-analysis templates`
7. `Add model card and dataset card`

Each commit should include either code, an experiment result, or a written research note.
